Є два види встановлення. Як встановити лише 1.
В git команда:
    node -v 
    щоб дізнатися версію node і чи встановлений він.

    npm -v
    Тепер те саме з npm.

    npm install -g typescript
    для встановлення глобально typescript.

    tsc -v
    дізнатися версію TS.
