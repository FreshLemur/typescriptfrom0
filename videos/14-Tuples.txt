Кортежі. Tuples - кортежі.
Кортежі - це структури даних, які дозволяють зберігати фіксовану кількість елементів різних типів в певному порядку.

Приклад простого кортежу:
    let user: [string, number, boolean];
    user = ["st", 1231, true];
    Ми створюємо масив user, який має мати такий самий формат, який ми створили.

Ще один приклад простого кортежу:
    let rgb: [number, number, number] = [255, 255, 255];

Приклад кортежу, використовуючи type, а також зі зміною:
    type User = [string, number];
    let newUser: User = ["Hello", 12131];
    newUser[1] = 0;
    В цьому випадку ми використовуємо кортеж за допомогою type, а також змінюємо елемент з індексом 1, що є
        number і змінюємо його значення на 0.

Приклад push:
    type User = [string, number];
    let newUser: User = ["Hello", 12131];
    newUser[1] = 0;
    newUser.push(true)
    В старих версіях TS він не бачить помилки, в мене найновіша і TS бачить помилку, тому що я намагаюся додати тип,
        якого не існує.
