Приклад надання строгої типізації того, що повертає фукнція:
    function returnString(s: string): string {
        return "";
    }
    Ця функція має обов'язково повернути рядок.

    Приклад з стрілковою функцією:
        const returnStringArrowFunction = (s: string): string => {
            return "";
        };

    Приклад з циклом та стрілковою функцією:
        const heros = ["thor", "spiderman", "ironman"];
        heros.map((hero): string => {
            return `hero is ${hero}`;
        });
        Тут також на виході ми повинні повернути рядкове значення, тобто string.
        TS достатньо розумний, щоб зрозуміти, що вхідне значення може бути тільки string, тому що в нашому масиві є лише string.

    Приклад з фукнцією, яка нічого не повертає:
        function consoleError(errmsg: string): void {
            console.log(errmsg);
        }
        Ця функція не повертає нічого і void слугує як маркер для іншого розробника, що ця функція нічого не повертає.

    Також існує never, проте я не дуже зрозумів його сенс. Про нього можна почитати в документації.
